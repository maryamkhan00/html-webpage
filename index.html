HTML

htmlis a presentation language interpreted by our browser which is used to give structure to a webpage. it is the most basic step to learn or start web development, it defines the meaning of a webpage. the current version of html is html 5 which has replaced html 4 due to which users can access many new html 5 elements and markups but html 5 does not support attributes which can be used to design or for presentation which is commonly done from css. HTML stands for hypertext markup language from which;

hypertext refers to the links used to connect webpages together within a single or multiple website.

andmarkupsare used to annotate text, images and other content for display in a browser which is done by using special elements such as;

  • <header>,
  • <main>,
  • <footer> etc.

These elements can also have attributes with them. attributes always contain some kind of value with them and are written in the following format;

attribute name="attribute value"

types of html markups

html markups can be divided into two catagories

  1. structural markups
  2. semantic markups

1-structural markups

html uses these markups to give structure to the webpage each section of the page for example a new para or a new heading even a subheading is enclosed in a greater than and a less than symbol <p> called tabs like so and then some kind of text is written inside it (the tabs along with the text inside it is together known as an element). structural markups effect how the content is displayed in a webpage.

2-semantic markups

these markups are used to help the screen reader and the server and they mostly don't have any effect on the content displayed within a webpage. the latest version of html is html 5. html 5 released some new elements and semantic markups like;

1. <strong>
used to identify that the text inside this tag has great value also appearance wise it bolds the text inside it.
2. <emphasis>
used to emphasize an important piece of information also appearance wise it makes the text inside the tag itallic.
